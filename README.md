# yesarly

Looking back your awesome year

```
$ ./yesarly --help
Usage of ./yesarly:
  -apikey string
        esa api key
  -team string
        esa team
  -user string
        esa username
  -from string
        from (format: 2020-01-01) (default: 1/1 of this year)
  -to string
        to (format: 2020-12-31) (default: 12/31 of this year)
```
